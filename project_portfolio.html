Contact Management

To run the program, you need to choose a menu option from 1-8. Then, depending on your choice you input a name and whichever contact type you want to add or change. This contact type needs to be spelled exactly as one of the following: "Phone number", "Email", "Additional info" or else you will get flagged in the program. To export a contact list into a file, please use the "export.txt" file as your file name. To import contacts into your list, please use the "import.txt" file as your filename.
